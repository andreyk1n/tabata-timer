# 🕒 Tabata Timer

Сучасний веб-додаток для інтервальних тренувань (Tabata / HIIT).  
Налаштовуй свій ритм, запускай таймер і просто працюй 💪

---

## 🚀 Features

- ⏱ **Гнучке налаштування таймера**
  - Work (робота)
  - Rest (відпочинок)
  - Round Reset (перерва між раундами)
  - Exercises (кількість вправ)
  - Rounds (кількість раундів)

- 🔁 **Автоматичний цикл тренування**
  - Work → Rest → наступна вправа → наступний раунд

- 🔔 **Звукові сигнали**
  - Відлік останніх секунд
  - Перехід між фазами

- 📱 **Адаптивний дизайн**
  - Mobile-first
  - Desktop preview з info-панеллю

- 🎯 **Візуальний прогрес**
  - Круговий таймер
  - Поточна фаза
  - Наступна вправа

- 📊 **Статистика**
  - Загальний час тренування
  - Кількість завершених тренувань (localStorage)

- 🔋 **Wake Lock API**
  - Екран не вимикається під час тренування
